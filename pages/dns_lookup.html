DNS Lookup

Show Description
Resolve DNS records for any domain. Supports multiple record types including A, AAAA, MX, TXT, CNAME, NS, SOA, and more.

Instructions:
  1. Enter a domain name (e.g., example.com)
  2. Select the DNS record type(s) you want to query
  3. Click "Lookup DNS" to resolve the records
Note: This tool uses public DNS-over-HTTPS (DoH) services for privacy and security.
