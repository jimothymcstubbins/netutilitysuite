Quick RDAP Lookup

Show Description

Extracts the TLD from a domain you input and uses https://data.iana.org/rdap/dns.json to find its domain registry. From the registry, extracts the domain's registrar.

Sends a lookup request to the registrar, falling back to the registry if the registrar returns no usable data, then parses the received data to extract the following:

  • Registrant Name(fn), Org, Address, Email
  • Administrative Name(fn), Org, Address, Email
  • Technical Name(fn), Org, Address, Email
  • Registrar Name
  • Dates - Created On, Expiration, Updated On
  • Nameservers
Please note: IANA only has RDAP information for TLD registries who provide information to IANA. Any registry IANA doesn't have information about will return an error.
Also note: While RDAP formats are standardized, not every registrar follows them the same way. Certain TLDs will return no information here because of parsing issues. Use the Full RDAP Lookup for these.
