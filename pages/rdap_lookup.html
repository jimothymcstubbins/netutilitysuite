RDAP Lookup

Show Description
Extracts the TLD from a domain and uses IANA's RDAP bootstrap to find the domain registry. Queries both registry and registrar RDAP endpoints to provide comprehensive domain registration data.

Quick Mode: Shows parsed contact information, dates, and nameservers in a readable format.
Full Mode: Displays the complete raw RDAP JSON response for advanced analysis.

Please note: IANA only has information for registrars who use RDAP. Some registrars may not return complete data.
