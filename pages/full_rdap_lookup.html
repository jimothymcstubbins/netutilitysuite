Full RDAP Lookup

Show Description

Extracts the TLD from a domain you input and uses https://data.iana.org/rdap/dns.json to find its domain registry. From the registry, extracts the domain's registrar.

Sends a lookup request to the registrar, falling back to the registry if the registrar returns no usable data, but does not parse any data.

Please note: IANA only has information for registrars who use RDAP. Any registrar IANA doesn't have information about will return an error.
